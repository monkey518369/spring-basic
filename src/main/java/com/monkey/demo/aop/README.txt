1.通知（Advice）
  切面所要完成的工作，比如打印日志，安全检查，缓存，事务管理。确定了在什么时候执行什么操作。
  前置通知（Before），后置通知(After)，环绕通知(Around)，返回通知(After-Returning)，异常通知(After-throwing)

2.连接点（Join point）
  所有可知运行通知的地方都称为连接点

3.切点（Pointcut）
  特定的执行通知的连接点。确定了在什么地方执行通知

4.切面（Aspect）
   通知和切点的结合，总称为切面，什么时候在什么地方执行什么操作

5.引入（Introduction）
  引入允许我们向现有类中添加新的方法或属性。

6.织入（Weaving）
  织入是把切面应用到目标对象生成代理对象的过程。
  可以分为编译器织入，类加载期织入，运行期织入
